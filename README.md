## ✨ 기능
- ✅ 소환사 검색 (이름#태그 형식)
- ✅ 소환사 기본 정보 표시 (레벨, 티어)
- ✅ 최근 3시즌 티어 정보
- ✅ 최근 10게임 AI Score 조회
- ✅ 평균 AI Score 자동 계산
- ✅ 반응형 디자인 (모바일 지원)
- ✅ 깔끔한 UI/UX

게임별 AI Score: 게임 1~10의 개별 점수
